
# Image Sharing App

website to share image

## Backend

To deploy this project run

```bash
  cd backend
```
```bash
  npm install
```
```bash
  docker build -t backend .
```
```bash
  docker compose up
```

## Frontend

To deploy this project run

```bash
  cd frontend
```
```bash
  npm install
```
```bash
  npm run dev
```
